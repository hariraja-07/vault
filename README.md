# vault

`vault` is a simple CLI-based key-value storage tool written in Go.
It allows you to set, get, delete, and list key-value pairs with JSON-based persistence.

---

## Usage

### Set a value:
```bash
vault set <key> <value>
```

### Get a value:
```bash
vault get <key>
```

### Delete a value:
```bash
vault delete <key>
```

### List all values:
```bash
vault list
```
